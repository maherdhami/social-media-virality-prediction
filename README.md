# Social Media Virality Prediction & Caption Intelligence

## Overview

This project develops an NLP-powered system for predicting social media post virality and analyzing caption effectiveness.

By combining engagement metrics and textual content analysis, the system identifies patterns associated with highly engaging posts and provides insights for content optimization.



## Problem Statement

Content creators and marketers often struggle to understand which factors contribute to viral content.
This project aims to analyze social media posts and predict engagement potential using machine learning and NLP techniques.


## Dataset

A synthetic dataset was created to simulate realistic social media posts and engagement patterns.

### Features Included

- Caption Text
- Topic Category
- Hashtag Count
- Post Length
- Likes
- Shares
- Comments
- Viral Label
